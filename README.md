# Deepfake Image Detection

This project detects whether a face image is **Real or Fake (Deepfake)** using a deep learning model.

## Model
MobileNetV2 (Transfer Learning)

## Technologies
- Python
- TensorFlow / Keras
- MobileNetV2
- Google Colab

## Dataset
140K Real and Fake Faces Dataset  
https://www.kaggle.com/datasets/xhlulu/140k-real-and-fake-faces

## Features
- Image preprocessing
- Transfer learning
- Binary classification
- Real vs Fake face detection
